Memory system for real agent workflows

MemLayer хранит решения, ограничения, риски, артефакты и reuse-paths в одном operational memory contour.

Это контур памяти для агентных циклов, которым мало временного prompt buffer: поиск, relevant context, imports, quality gates, review queues, lifecycle cleanup, compaction и operator visibility.

Что делает MemLayer

Retrieval

Поиск по проекту, связанным проектам и глобальному контуру с lexical, semantic и hybrid режимами.

Quality

Quality scoring, semantic duplicate signals, operator review actions и lifecycle maintenance.

Imports

Структурный import, reimport, conflict detection и doc-aware ingestion для реальных репозиториев.

Operations

Runtime self-check, metrics, review queues, task logs, compaction и встроенная admin console.

Публичные точки входа

Публичный контур намеренно небольшой. Production API живёт на отдельном поддомене. Admin runtime вынесен отдельно и не рекламируется с этой страницы.

Health check 
curl -sS 'https://api.memlayer.ru/health'

Runtime self-check

Для лёгкой read-only проверки есть отдельный endpoint. Используй собственный API key и не храни его в shell history, если это не нужно.

Read-only self-check 
curl -sS 'https://api.memlayer.ru/admin/runtime/self-check?search_query=architecture&limit=5' \
  -H 'Authorization: Bearer YOUR_API_KEY'

Memory quality layer

MemLayer не просто хранит знания. Он проверяет, остаются ли они полезными: placeholder rejection, semantic duplicate signals, review queues, conflict resolution и compaction встроены в runtime.

  • Quality checks на write и update flows
  • Semantic duplicate signals внутри проекта
  • Decision authority и operator conflict resolution
  • Lifecycle decay, archive и compaction passes

Import и reuse

Project scan умеет поднимать память из реальных репозиториев, включая hidden docs, handoff files и структурированные артефакты. После этого retrieval можно ограничивать проектом, соседними проектами или глобальным контуром.

Project search 
curl -sS 'https://api.memlayer.ru/memory/search?query=architecture%20decision&mode=hybrid&limit=8' \
  -H 'Authorization: Bearer YOUR_API_KEY'
